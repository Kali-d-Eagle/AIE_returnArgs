üîπ 4. Return Fn (Wid Args)

Ye fn aise hota h jisme i/p args hote h aur fn ek result return krta h.
"Return" ka mtb hai: fn ek value return krega.
"Wid Args" ka mtb h: fn ko kuch i/p pass kia jayega, aur fn un i/ps ko process kr ke ek result dega!

üí° Real-Life Analogy:
Sochiye aap ek custom pizza order kar rahe ho.

üçï Pizza Order (fn call wid args): Aap apne pizza ke liye toppings, size, aur crust type select karte ho ‚Äî yeh sab args hain jo aap fn ko pass karte ho.
üç¥ Pizza Shop (fn processing) : Pizza shop in args ko use karke aapke hisaab se pizza tayar krta h.
üõµ Pizza Delivery (Return value): Jab pizza tayar ho jata h, shop aapko aapke custom pizza ko return krti h.
‚úÖ Aapko custom pizza milta h, jo aapne jo details diye the usi ke hisaab se.

Yahi hota h return fn wid args:
  • üçï Pizza Order = fn call with args
  • üìù Toppings, size, crust = i/p args
  • üç¥ Pizza Shop = fn processing the inputs
  • üõµ Delivery = return value (custom pizza)

üîß Syntax (Structure)

returnType fnName(argumentType argumentName) {
      // fn ka kaam
      return result;
    }

Breakdown:

  • returnType: fn ka return type, jo return kiya jayega (e.g., int, float)
  • fnName(): fn ka naam aur parentheses ke andar i/p args (i/p values)
  • { }: fn ka kaam yha likajata h
  • return result;: fn ek value return krta h based on inputs
Use Kab Karein?
Jab aapko kisi task ko perform karte wqt kuch inputs dene ho aur us task ka output ya result chahiye ho.
